## 🚀 GUÍA DE INICIO RÁPIDO

### 1. LOCAL (Sin servidor)
```bash
# Simplemente abre index.html en el navegador
```

### 2. CON SERVIDOR LOCAL
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx http-server
```

### 3. DESPLIEGUE EN NETLIFY
```bash
# Método Drag & Drop o
netlify deploy --prod
```

### 4. DESPLIEGUE EN VERCEL
```bash
npm install -g vercel
vercel --prod
```

## 🛠️ PERSONALIZACIÓN

### Cambiar Colores
Edita `:root` en `styles.css`:
```css
:root {
  --primary-brown: #TU_COLOR;
  --accent-orange: #TU_COLOR;
  --accent-green: #TU_COLOR;
}
```

### Cambiar Contenido
1. Abre `index.html`
2. Edita textos directamente
3. Actualiza `config.json` con datos de la empresa

### Agregar Nuevas Secciones
```html
<!-- Copiar sección y personalizar -->
<section class="nueva-seccion">
  <h2 class="section-title">Título</h2>
  <p class="section-subtitle">Subtítulo</p>
  <!-- Contenido -->
</section>
```
